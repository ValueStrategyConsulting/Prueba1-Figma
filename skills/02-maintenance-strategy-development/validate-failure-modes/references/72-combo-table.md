## Naming Rules Reference

### FM-01: Failure Mode "what" Field
- Must start with a capital letter.
- Valid: "Corroded internal surface causing wall thinning"
- Invalid: "corroded internal surface" -> raises ValueError

### FM-02: Hidden Failure Consequence Consistency
- is_hidden=True -> consequence must be HIDDEN_SAFETY or HIDDEN_NONSAFETY
- is_hidden=False -> consequence must NOT be HIDDEN_*

### T-17: Task Constraint vs Access Time
- ONLINE constraint -> access_time_hours must be 0
- OFFLINE constraint -> access_time_hours must be > 0

### Task Name: Max 72 characters (SAP short text limit)

### WP-06: Work Package Name
- Must be ALL CAPS, max 40 characters
- Valid: "PUMP OVERHAUL PM02 WEEKLY"
- Invalid: "Pump Overhaul PM02 Weekly" -> raises ValueError

### Operation Numbers: Must be multiples of 10 (SAP convention)
