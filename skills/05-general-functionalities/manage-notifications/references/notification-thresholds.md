# Notification Thresholds Reference

## Alert Types and Severity

| Type | Trigger Condition | CRITICAL Threshold | WARNING Threshold |
|------|-------------------|-------------------|-------------------|
| RBI_OVERDUE | next_inspection_date <= today | > 90 days overdue | <= 90 days overdue |
| KPI_BREACH | value < target * 0.9 OR BELOW_TARGET | value < target * 0.7 | value < target * 0.9 |
| EQUIPMENT_RISK | health score <= threshold | score <= 30 | score <= threshold, > 30 |
| BACKLOG_AGING | age >= aging_threshold_days | age >= threshold * 3 | age >= threshold |
| CAPA_OVERDUE | OPEN/IN_PROGRESS past target_date | Always CRITICAL | N/A |
| MOC_OVERDUE | DRAFT/SUBMITTED/REVIEWING > 30 days | N/A | Always WARNING |

## Default Thresholds

| Parameter | Default Value |
|-----------|---------------|
| aging_threshold_days | 30 |
| health_threshold | CRITICAL (30) |
| Available health thresholds | CRITICAL: 30, AT_RISK: 50, FAIR: 70 |
| MoC stall threshold | 30 days |

## Notification Levels

| Level | Description |
|-------|-------------|
| CRITICAL | Immediate action required |
| WARNING | Attention needed, not yet critical |
| INFO | Informational (not currently generated by engine) |

## Reliability KPI Thresholds

| KPI | Target | Direction | Alert Trigger |
|-----|--------|-----------|---------------|
| availability_pct | 90.0% | Higher better | < 90.0 * 0.9 = 81.0% |
| reactive_ratio_pct | 20.0% | Lower better | > 20.0 * 1.1 = 22.0% |

## CAPA Status Filter

Only these statuses are checked for overdue:
- OPEN
- IN_PROGRESS

These are excluded (never flagged as overdue):
- CLOSED
- VERIFIED

## MoC Status Filter

Only these statuses are checked for stalling:
- DRAFT
- SUBMITTED
- REVIEWING
